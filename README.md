# Master Console

[Master](https://github.com/Ken-Chy129/master) 框架的前端管理控制台，基于 Ant Design Pro 构建，提供可视化的运行时变量管理界面。

## 功能特性

- **应用管理**：查看已接入的应用及其机器状态
- **变量管理**：在线查询和修改应用运行时变量
- **场景模板**：创建可复用的变量配置模板，一键批量下发
- **变更日志**：记录所有变量修改历史，支持追溯

## 技术栈

- React 18
- Ant Design Pro / Ant Design 5
- UmiJS Max
- TypeScript

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm start:dev

# 构建生产版本
pnpm build
```

需要配合 [Master 服务端](https://github.com/Ken-Chy129/master) 一起使用。
